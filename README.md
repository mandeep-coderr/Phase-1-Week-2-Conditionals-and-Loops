## 4. Вложенные циклы (`nested_loops.py`)

Файл демонстрирует ключевые концепции вложенных циклов в Python.

### Примеры

- **Отжимания по подходам** — внешний цикл перебирает подходы (1–3), внутренний — отжимания (1–5).
- **Заказы пиццы** — внешний цикл запускает заказ, внутренний выпекает каждую пиццу (1–3).
- **Временна́я сложность** — наглядный подсчёт итераций вложенного цикла:
  - Для `n` элементов вложенный цикл выполняет `n²` операций.
  - Пример: 10 элементов → 100 действий.
- **`break` во вложенном цикле** — прерывание внутреннего цикла при выполнении условия (пример: тигр в комнате 3).

### Ключевые концепции

- Внешний цикл выполняется один раз → внутренний цикл проходит полностью.
- Оператор `break` прерывает **только внутренний** цикл; внешний продолжает работу.
- Временна́я сложность вложенного цикла: **O(n²)**.
