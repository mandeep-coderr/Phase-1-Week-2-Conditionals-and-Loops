## Темы

### 1. Условные операторы (`if / elif / else`)
**Файл:** `1. if_elif_else_conditions/if_elif_else.py`

Примеры закомментированы и разбиты по темам:
- `if` — простая проверка условия.
- `else` — ветка при ложном условии.
- `if / else` — двусторонняя ветка (пример с температурой).
- `if / elif / else` — многоуровневая ветка (пример с оценками).
- **Отступы (indentation)** — правила оформления блоков кода.

### 3. Цикл `for` и `range()`
**Файл:** `3. for_Loop_and_range()/for_loop_range.py`

- **`for` по списку** — перебор элементов один за другим.
- **`range(stop)`** — генерация чисел от 0 до stop.
- **`range(start, stop)`** — с заданным началом.
- **`range(start, stop, step)`** — с шагом, в том числе отрицательным (обратный отсчёт).
- **Перебор строки** — посимвольный обход строки.
- **`enumerate()`** — получение индекса и значения одновременно.
- **`for-else`** — блок `else` выполняется, если цикл завершился без `break`.
