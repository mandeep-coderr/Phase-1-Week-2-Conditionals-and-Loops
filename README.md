# Phase 1 – Week 2: Conditionals and Loops

A collection of Python exercises covering conditionals and loops.

## Topics Covered

- **Mixing Conditions & Loops** (`5. Mixing_Conditions_Loops/mixing_loops_conditions.py`): demonstrates `if/elif/else` inside `for` loops, grade checking, the accumulator pattern (e.g. counting error logs), and flag variables (e.g. triggering an alarm on a specific condition).
