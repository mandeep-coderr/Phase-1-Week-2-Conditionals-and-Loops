# Phase 1 – Week 2: Conditionals and Loops

A beginner Python practice repository guided by a Socratic / Senior Python Mentor approach.

## Topics Covered

- **if / elif / else conditions** (`if_elif_else_conditions/if_elif_else.py`)
  - Basic `if`, `if/else`, and `if/elif/else` branching
  - Nested `if` statements (castle/dragon adventure example)
  - Ternary (one-line) expressions: `value = X if condition else Y`
  - Real-world example: Free vs. Pro user subscription label

- **While loops** (`While_loop/while_loop.py`)
  - Basic counter loop (push-ups example)
