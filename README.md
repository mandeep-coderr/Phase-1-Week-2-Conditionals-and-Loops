# Phase 1 – Week 2: Conditionals and Loops

A collection of Python exercises and mini-projects covering conditionals and loops.

## Contents

- **2. While Loop** (`While_loop/while_loop.py`): Demonstrates `while` loop fundamentals including loop variables, countdown timers, and `break`/`continue` usage. Earlier examples are retained as commented-out reference code.

- **6. Mini Project – CLI Auth Lockout System** (`6. Mini project- CLI Auth Lockout System/CLI_auth_lockout_system.py`): A command-line PIN authentication system that:
  - Allows up to **3 attempts** to enter the correct PIN (`1234`).
  - Handles empty input gracefully.
  - Locks the account (blocks the ATM card) after all attempts are exhausted.
  - Grants access immediately upon a correct PIN entry.
