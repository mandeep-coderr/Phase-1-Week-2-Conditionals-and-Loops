## Модули

### 2. While_loop/while_loop.py

Демонстрирует базовые концепции цикла `while` в Python.

**Темы:**

- **Переменная цикла (Loop Variable)** — счётчик, управляющий итерациями.
- **Цикл `while`** — выполняет тело, пока условие истинно.
- **Обновление переменной цикла** — предотвращает бесконечный цикл.
- **Оператор `break`** — досрочный выход из цикла при выполнении условия.

**Примеры из файла:**

| Пример | Описание |
|---|---|
| `items_packed` | Упаковка 3 предметов; цикл завершается по счётчику |
| `rocket_timer` | Обратный отсчёт от 10 до 0 |
| `search_number` + `break` | Перебор чисел 1–5 с остановкой при достижении 4 |
